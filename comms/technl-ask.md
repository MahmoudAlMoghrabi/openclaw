# What we need from TechNL to lock everything in

A few confirmations so we can size and finalize:

- **Headcount, and solo or pairs.** This sets how many API keys we generate and
  how we spread them across projects for rate-limit headroom.
- **Session length:** 60, 90, or 120 minutes.
- **Room check:** can the wifi comfortably handle everyone running a browser-based
  tool at the same time? Enough power outlets? A screen or projector for the demos
  at the end?
- **Laptops:** are attendees bringing personal or work laptops? Work laptops
  sometimes block browser dev tools, so a heads-up helps us prepare.
- **Gemini keys:** we need one dedicated Google account that TechNL controls
  (not anyone's personal account) with billing enabled. Expected usage is
  roughly $15-30 for 50 attendees; we cap the account at $100 and Google
  independently enforces a $250 ceiling, so cost cannot run away. Whether
  TechNL creates the account and hands us access, or funds one we set up, is
  your call. We revoke every key and disable billing right after the session.
  Send us the headcount and we will size the rest.
